# todo-cli

TODO-CLI CRUD application with TypeScript, C++, C#, Go, and Rust

# cpp

Installation:

```
mingw32-make
```

To Run:

```
./main
```

# typescript

Installation:

```
npm install
```

To Run:

```
npm run start
```

# cs

Build:

```
dotnet build
dotnet run
```

Format:

ctrl + k, ctrl + d
