# todo-cli

TODO-CLI CRUD application with various different languages

# /ts

```
npm run start
```

# /go

```
go run .
```

_Change the GOTMPDIR env variable and exclude that path in your firewall if issues running a tmp exe occurs_

# /cs

```
dotnet build .
```

```
dotnet run
```

# /py

```
python app.py
```

# /rs

```
cargo build
```

```
cargo run
```
