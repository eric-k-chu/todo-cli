# todo-cli

TODO-CLI CRUD application with TypeScript, C++, C#, Go, and Rust

# cpp

Installation:

- Download the `json.hpp` file from https://github.com/nlohmann/json
- Place it in `/cpp/nlohmann/json.hpp`

# typescript

Installation:

```
npm install
```

To Run:

```
npm run start
```

# cs

Build:

```
dotnet build
dotnet run
```

Format:

ctrl + k, ctrl + d
