# todo-cli

TODO-CLI CRUD application with TypeScript, C++, C#, Go, and Rust

# /ts

```
npm run start
```

# /go

```
go run .
```

# /cpp

`json.hpp` _from https://github.com/nlohmann/json must be downloaded and placed into /cpp_

Compile:

```
g++ -std=c++17 app.cpp -o app
```

```
./app.exe
```
